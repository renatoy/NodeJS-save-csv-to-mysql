# Node.js Rest APIs with csv file saved to db

## First
Set up your db credentials in ``app/config/db.config.js``

### Setup
```
npm install
```

### Run
```
npm start
```
### Or
```
npm i && npm start
```



### Important:
To actually insert customers in db, you must uncomment from line 36 in /server.js
Then comment again and:
$ npm start
